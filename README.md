## 💲 로또번호 발급 사이트 💲
<br>
<br>

| 항목 | 내용 |
| ------ | ------ |
| 참여인원 | 1명 |
| 개발기간 | 2022.10.17 ~ 2022.11.04 (약 3주 소요) |
| 개발목표 | Java, JavaScript, JSP, Bootstrap을 이용하여 랜덤번호 발급, 게시판 등을 개발 및 구현합니다. |
| 구현설명 | 메인화면에서 번호 받기 클릭 시 랜덤번호 출력 / 다양한 게시판 구현 |
| 담당업무 | 인트로 페이지 제작 / 로그인, 회원가입 페이지 제작, 게시판 제작, 메일 보내기 |

<br>
